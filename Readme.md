# Netflix Data Analysis using SQL

## 📌 Project Overview
This project analyzes Netflix dataset using SQL to extract insights about content trends, genres, and production.

## 🛠 Tools Used
- MySQL Workbench
- SQL

## 📊 Dataset
Netflix Movies and TV Shows dataset from Kaggle

## 🔍 Key Analysis
- Movies vs TV Shows distribution
- Content growth over years
- Top countries producing content
- Genre analysis
- Ratings distribution

## 📈 Key Insights
- Most content added after 2015
- Movies dominate over TV Shows
- USA produces maximum content
- Drama and International genres are most common

## 📁 Project Structure
- dataset/ → raw data
- sql/ → SQL queries
- README.md → project documentation

## 🚀 Conclusion
This project demonstrates SQL skills including joins, aggregations, and data analysis to derive business insights.
